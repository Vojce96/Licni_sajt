Opis izabrane tehnologije

Opis izabrane tehnologije je opisan u seminarskom radu koji se nalazi ispod

jQuery UI Izabrana tehnologija je jQuery UI. jQuery UI je kolekcija GUI vidžeta, animiranih vizuelnih efekata i tema implementiranih sa jQuery (JavaScript bibliotekama), Cascading Stile Sheets i HTML. Prema JavaScript analitičkom servisu, Libscore, jQuery UI se koristi na preko 197.000 od prvih milion sajtova, što ga čini drugom najpopularnijom JavaScript bibliotekom. Značajne korisnike čine Pinterest, PayPal, IMDb, The Huffington Post i Netflix.I jQuery i jQuery UI su besplatni i open-source softveri distribuirani od strane jQuery Fondacije pod MIT licencom; jQuery UI je prvi put objavljen u septembru 2007. godine.
